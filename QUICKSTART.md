# 🎮 COUNTRY BOX v2.0 - QUICK START

## 🚀 Chạy Game Nhanh

### 1. Textual UI (Khuyến nghị - TUI hiện đại)
```bash
python launch.py
# Sau đó chọn 1
```

Hoặc chạy trực tiếp:
```bash
python run_textual_new.py
```

**Giao diện:**
- 🗺️ **Map View**: Viewport 80x20 của map 90x75
- 📊 **Status Panel**: Real-time game status
- 💬 **Command Input**: Gõ lệnh và Enter
- 📝 **Message Log**: Event history

### 2. CLI Mode (Dòng lệnh thuần)
```bash
python run_cli.py
```

### 3. Demo (Xem trước)
```bash
python demo_textual.py
```

### 4. Test Suite
```bash
python test_textual_ui.py
```

## 🎛️ Cách Chơi Cơ Bản

1. **Khởi động game** → Chọn Textual UI
2. **Xem status** → Gõ `status` + Enter
3. **Xem map** → Gõ `map` + Enter
4. **Điều khiển dân làng**:
   ```
   hey_village find * wood     # Thu thập gỗ
   hey_village make * food     # Làm thức ăn
   ```
5. **Theo dõi resources** → Đảm bảo đủ thức ăn/nước
6. **Chế tạo công cụ** → Xây smelter, làm axe/steel

## 🎯 Mục tiêu
- Giữ dân làng sống sót (đói/khát < 100)
- Phát triển làng với công cụ tốt hơn
- Khai thác tài nguyên hiệu quả
- Sinh tồn qua nhiều lượt

## 💡 Pro Tips
- Theo dõi hunger/thirst của từng villager
- Khai thác wood/stone trước để làm công cụ
- Xây smelter để luyện iron → steel
- Mua bán với merchants khi xuất hiện
